# Complect - Frontend

## Run in Docker Container
1. Start dev server: `docker-compose up`
2. Navigate to `localhost:81`

## or Run locally
1. Install dependencies: `npm i`
2. Start dev server: `npm run dev`
3. Navigate to `localhost:81`
